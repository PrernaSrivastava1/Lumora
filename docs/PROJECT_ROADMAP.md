# Project Roadmap

This document outlines the milestones and future plans for the Lumora engine.

## Phase 1: Core Indexing (Completed)
- [x] Brute-force exhaustive search
- [x] KD-Tree space partitioning index
- [x] Hierarchical Navigable Small World (HNSW) graph index
- [x] Dynamic AUTO routing strategy

## Phase 2: Production Readiness (Completed)
- [x] REST API versioning (`/api/v1`)
- [x] Environment-specific configurations and active profiles
- [x] One-command Windows startup wrapper (`start-dev.ps1`)
- [x] Comprehensive automated tests (75/75 green)

## Phase 3: Advanced Vector Operations (Future)
- [ ] Real-time HNSW graph traversal visualizer (using HTML5 Canvas)
- [ ] Metadata filtering alongside distance calculations
- [ ] Model quantization configurations for memory reduction
